# Notes

## Naming conventions

- Folder/repository names: `kebab-case`
    - Example: `python-for-ai-source-code`
    - Note: this is my personal preference; some developers use `snake_case` for folder names.
- Python naming standard: `snake_case`
    - This is the standard convention in Python.
    - Use for folders, files, functions, variables, and similar code names.
    - Examples: `week_01_intro`, `capital_tax`, `run_calculator`
